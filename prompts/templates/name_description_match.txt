 - A service/expense/charge is NOT a physical asset. It is a cost, fee, or labor entry.
 - If the asset name contains ANY of these keywords (alone or combined with an asset name), it is NOT a valid asset -> **REJECT immediately**:
 "installation charges", "installation cost", "service charges", "service cost", "labor charges", "labor cost",

 - CRITICAL: Even if an asset name is PART of the entry (e.g., "AC Installation Charges", "Laptop Repair Cost"), the ENTIRE entry is still a service/expense -> **REJECT**.
 - The key test: "Can you physically tag this with a barcode sticker?" If NO -> REJECT.

 Examples of NON-ASSETS (REJECT all):
 - "AC Installation Charges" -> REJECT (installation service, not an AC unit)
